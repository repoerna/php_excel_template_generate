## Export Data To Ecel Template using PHP
### Requirement
#### - php zip extension
    windows:
    - https://www.php.net/manual/en/install.pecl.windows.php 
    ubuntu:
    - sudo apt-get install php-zip

#### - PHPSpreadsheet pakage
    -   composer require phpoffice/phpspreadsheet --prefer-source
    
#### - Excel Template
    
### Run php
    - php generate.php
